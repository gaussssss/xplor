# Fonctionnalites implementees

- Architecture Clean + MVVM avec separation data/domain/presentation.
- Navigation de fichiers locale avec liste/grille, breadcrumbs, barre de chemin et favoris/systeme.
- Recherche par nom/extension sur le dossier courant.
- Selection multi (clic, clic long), menu contextuel au clic droit, ouverture par double-clic, drag & drop vers un dossier.
- Actions: creation de dossier, copie/coller, couper/coller, duplication, renommage, suppression (actuelle suppression directe), deplacement vers un autre chemin, affichage dans Finder, rafraichissement.
- Affichages etats: chargement, erreur avec retry, vide (y compris acces limite).
- Theming sombre futuriste et composants reutilisables (GlassPanel, ToolbarButton, SidebarSection, BreadcrumbBar, FileEntryTile).
